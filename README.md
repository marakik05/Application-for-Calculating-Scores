# Application-for-Calculating-Scores
This is a desktop application that allows the calculation of scoring for farmers wishing to participate in Measure P3-73-2.9 – Investments for the modernization and construction of greenhouses and accessible plant production shelters, as part of the Strategic Plan of the Common Agricultural Policy of Greece 2023–2027
